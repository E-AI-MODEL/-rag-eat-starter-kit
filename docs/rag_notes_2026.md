# RAG Notes 2026

## Purpose

Starter notes for Retrieval-Augmented Generation projects.

## Basic flow

1. Collect approved sources.
2. Clean documents.
3. Split documents into useful chunks.
4. Add metadata.
5. Build a search index.
6. Retrieve candidate passages.
7. Rerank candidates.
8. Answer with source references.
9. Test with fixed questions.
10. Review feedback.

## Recommended first setup

- hybrid retrieval
- metadata per chunk
- reranking
- source references
- fixed evaluation set

## Evaluation

Check source recall, answer quality, citation quality and handling of missing context.
