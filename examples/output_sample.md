# Output sample

sources do not support an answer, the assistant says so instead of guessing.

```text
Answer:
I cannot answer this reliably based on the retrieved sources.

Sources:
- None of the retrieved passages cover this question.

Uncertainty:
The available context does not contain a reliable passage that supports this question.
```
