# What this starter kit does

This starter kit answers questions from the documents in the knowledge folder. It should not guess when the answer is missing.

# How to try it

Start the web app, keep the corpus set to knowledge, and ask: What does this starter kit do?

# Safety rule

If the document does not contain the answer, the assistant should say that it cannot answer reliably from the retrieved sources.
